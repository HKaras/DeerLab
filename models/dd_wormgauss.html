dd_wormgauss¶

deerlab.dd_models.dd_wormgauss(r, param)[source]¶

Worm-like chain model near the rigid limit with Gaussian convolution

The function takes a list or array of parameters and returns the calculated distance distribution:

P = dd_wormgauss(r,param)

The built-in information on the model can be accessed via its attributes:

dd_wormgauss.parameters  # String list of parameter names
dd_wormgauss.units       # String list of metric units of parameters
dd_wormgauss.start       # List of values used as start values during optimization 
dd_wormgauss.lower       # List of values used as lower bounds during optimization
dd_wormgauss.upper       # List of values used as upper bounds during optimization
Parameters:
r : array_like

Distance axis, in nanometers.

param : array_like

List of model parameter values.
Returns:
P : ndarray

Distance distribution.

Notes

Model:

Variable Symbol Start Value Lower bound Upper bound Description
param[0] 3.7 1.5 10 Contour length (nm)
param[1] 10 2 100 Persistence length (nm)
param[2] 0.2 0.01 2 Gaussian standard deviation (nm)

References

[1]J. Wilhelm, E. Frey, Radial Distribution Function of Semiflexible Polymers Phys. Rev. Lett. 77(12), 2581-2584, 1996
