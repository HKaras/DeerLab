ex_ridme7¶

deerlab.ex_models.ex_ridme7(param)[source]¶

RIDME experiment model (spin S=7/2)

The function takes a list or array of parameters and returns the calculated experiment dipolar pathways:

pathways = ex_ridme7(param)

The built-in information on the model can be accessed via its attributes:

ex_ridme7.parameters  # String list of parameter names
ex_ridme7.units       # String list of metric units of parameters
ex_ridme7.start       # List of values used as start values during optimization 
ex_ridme7.lower       # List of values used as lower bounds during optimization
ex_ridme7.upper       # List of values used as upper bounds during optimization
Parameters:
param : array_like

List of model parameter values.
Returns:
pathways : ndarray

Dipolar pathways of the experiment

Notes

Model:

This experiment model has seven harmonic pathways and an unmodulated contribution. The kernel is

Variable Symbol Start Values Lower Upper Description
param[0] 0.3 0 1 Amplitude of unmodulated contribution
param[1] 0.5 0 1 Amplitude of 1st harmonic pathway
param[2] 0.3 0 1 Amplitude of 2nd harmonic pathway
param[3] 0.2 0 1 Amplitude of 3rd harmonic pathway
param[4] 0.1 0 1 Amplitude of 4th harmonic pathway
param[5] 0.05 0 1 Amplitude of 5th harmonic pathway
param[6] 0.02 0 1 Amplitude of 6th harmonic pathway
param[7] 0.01 0 1 Amplitude of 7th harmonic pathway
