ex_4pdeer¶

deerlab.ex_models.ex_4pdeer(param)[source]¶

Single-pathway 4-pulse DEER experiment model

The function takes a list or array of parameters and returns the calculated experiment dipolar pathways:

pathways = ex_4pdeer(param)

The built-in information on the model can be accessed via its attributes:

ex_4pdeer.parameters  # String list of parameter names
ex_4pdeer.units       # String list of metric units of parameters
ex_4pdeer.start       # List of values used as start values during optimization 
ex_4pdeer.lower       # List of values used as lower bounds during optimization
ex_4pdeer.upper       # List of values used as upper bounds during optimization
Parameters:
param : array_like

List of model parameter values.
Returns:
pathways : ndarray

Dipolar pathways of the experiment
