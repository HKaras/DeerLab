bg_sumstrexp¶

deerlab.bg_models.bg_sumstrexp(t, param)[source]¶

Sum of two stretched exponentials background model

Parameters:
t : array_like

Time axis, in microseconds.

param : array_like

List of model parameter values.
Returns:
B : ndarray

Dipolar background.

Notes

Model:

Variable Symbol Start Value Lower bound Upper bound Description
param[0] 0.25 0 200 1st strexp decay rate (μs-d)
param[1] 1 0 6 1st strexp stretch factor
param[2] 0.25 0 200 2nd strexp decay rate (μs-d)
param[3] 1 0 6 2nd strexp stretch factor
param[4] 0.50 0 1 Relative amplitude

The function takes a list or array of parameters and returns the calculated background model:

B = bg_sumstrexp(t,param)

The built-in information on the model can be accessed via its attributes:

bg_sumstrexp.parameters  # String list of parameter names
bg_sumstrexp.units       # String list of metric units of parameters
bg_sumstrexp.start       # List of values used as start values during optimization 
bg_sumstrexp.lower       # List of values used as lower bounds during optimization
bg_sumstrexp.upper       # List of values used as upper bounds during optimization
