dd_rice3¶

deerlab.dd_models.dd_rice3(r, param)[source]¶

Sum of three 3D-Rice distributions

The function takes a list or array of parameters and returns the calculated distance distribution:

P = dd_rice3(r,param)

The built-in information on the model can be accessed via its attributes:

dd_rice3.parameters  # String list of parameter names
dd_rice3.units       # String list of metric units of parameters
dd_rice3.start       # List of values used as start values during optimization 
dd_rice3.lower       # List of values used as lower bounds during optimization
dd_rice3.upper       # List of values used as upper bounds during optimization
Parameters:
r : array_like

Distance axis, in nanometers.

param : array_like

List of model parameter values.
Returns:
P : ndarray

Distance distribution.
